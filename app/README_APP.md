# Retail Forecasting & Sales Analytics Chatbot

This Streamlit app provides a chatbot-style interface for business users to ask natural-language questions about forecast outputs, restock recommendations, model performance, profit opportunities, and historical sales records. It reads local CSV files generated by the forecasting pipeline and does not retrain models when the app opens.

The app is read-only. It does not update Snowflake or write prediction data back to Snowflake.

## How To Run

From the project root:

```powershell
.\.venv\Scripts\Activate.ps1
streamlit run app/streamlit_app.py
```

If Streamlit is not installed:

```powershell
python -m pip install -r requirements.txt
```

## Expected Input CSVs

- `data/outputs/forecasts/revenue_forecast_next_30_days.csv`
- `data/outputs/forecasts/profit_forecast_next_30_days.csv`
- `data/outputs/forecasts/product_store_demand_forecast.csv`
- `data/outputs/forecasts/product_demand_forecast.csv`
- `data/outputs/restock/restock_recommendations_next_7_days.csv`
- `data/outputs/profit_opportunity/top_profit_products_next_30_days.csv`
- `data/outputs/profit_opportunity/top_profit_products_by_store_next_30_days.csv`
- `reports/model_evaluation_summary.csv`
- `data/processed/sales_cleaned.csv`
- fallback: `data/raw/sales_joined_raw.csv`

Missing files do not crash the app. The sidebar shows which datasets loaded successfully.

## Example Questions

- What is the expected revenue for the next 30 days?
- What is the expected gross profit for next month?
- Which products need restock this week?
- What are the top profit products?
- Which products have highest forecasted demand?
- How good is the model?
- What is total historical revenue?
- What are the top selling products historically?
- Show me sample historical records.

## Optional OpenAI Support

The core answers are rule-based and come from the local CSV files. If `OPENAI_API_KEY` exists in `.env`, the sidebar can enable an optional phrasing layer. It only sends the summarized answer text, not large datasets.

## Troubleshooting

If CSVs are missing, run the ML pipeline first:

```powershell
python main.py
```

If outputs are empty, check whether the Snowflake extraction and forecasting pipeline completed successfully.

If Streamlit is missing, install requirements:

```powershell
python -m pip install -r requirements.txt
```

If a column mismatch occurs, the app attempts to detect likely column names automatically and avoids showing generated index columns such as `Unnamed: 0` or `index`. Check the sidebar dataset status and inspect the CSV headers if a specific answer is unavailable.
